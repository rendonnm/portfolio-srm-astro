I led the first software project developed in-house for a client, driving the creation of the development area within the company.

I applied a feature-oriented architecture; managed global state with **Zustand**, routing with **React Router**, and server state synchronization with **TanStack Query**.

I implemented interfaces from design specifications, ensuring **visual consistency**, **accessibility**, and cross-device/cross-browser compatibility.

I ensured quality through **end-to-end (Playwright)** and **unit/component (Jest, React Testing Library)** tests.
